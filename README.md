# MongoDB_Shop

Za pokretanje aplikacije koriscena je docker verzija mongodb 5.0 koja je radjena na lab vezbi 2024/25. Html fajl se nalazi u folderu Front i moze se pokrenuti live serverom. Korisnik sa admin ulogom se kreira direktno u bazi. Konfiguracija za MongoDB se nalazi u Configurations folderu gde smo koristili konfiguraciju sa lab vezbi.
